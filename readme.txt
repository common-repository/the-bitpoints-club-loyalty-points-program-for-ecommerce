== Installation ==

1. Upload "BitPointsClub.zip" to the "/wp-content/plugins/" directory
or
Use the Wordpress Plugin search to install from within Wordpress
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In Wordpress, go to the BitPoints menu option.
1. Press the Auto Configure button
1. Save your changes
1. Place the shortcode on your cart and checkout pages
